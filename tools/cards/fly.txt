FLIGHT
ПОЛЕТ
Животное с этим свойством не может быть атакована хищником, у которого столько же или больше свойств.

fly
INTELLECT
ИНТЕЛЛЕКТ
Хищник с этим свойством один раз в ход может игнорировать одно из свойств животное, которое он собирается атаковать.

fly
METAMORPHOSE
МЕТАМОРФОЗА
Один раз за ход вместо взятия красный из кормовой базы, животное может отбросить одну из своих карт свойств (не приводящих к увеличению потребности в еде) и получить синий.

fly
VIVAPAROUS
ЖИВОРОЖДЕНИЕ 
Животное, став накормленным, производит новое животное - поместить верхнюю карту с колоды на поле как новое животное. В этот ход оно также считается накормленным.

fly
AMBUSH HUNTING
ЗАСАДА
Хищник с этим свойством может атаковать животное, когда оно получает красный из кормовой базы. Если атакованное животное съедено, красный остаётся в кормовой базе.

fly
TREMATODE
ТРЕМАТОДА
Паразит. Сыграть одновременно на пару животное животное другого игрока. Не может быть сброшена с помощью свойств других карт.

fly
ANGLERFISH
УДИЛЬЩИК
Можно сыграть только как животное. Когда Хищник атакует ваше животное, на котором нет свойств, удильщика можно перевернуть и вне очереди атаковать напавшего хищника, игнорируя одно из его свойств. При этом удильщик также не должен иметь каких-либо свойств.

fly
INK CLOUD
ЧЕРНИЛЬНОЕ ОБЛАКО
Один раз за фазу питания, когда атаковано хищником, животное может выбросить чернила, и не будет съедено. Хищник может снова атаковать в следующую свою фазу питания текущего хода это или другое животное.

fly
SHELL
РАКОВИНА
Если атаковано хищником, животное может спрятаться в раковину. В раковине животное не может быть съедено, но и до конца хода не может получать красный \ синий, кроме как преобразуя желтый. Атаковавший Хищник не может более атаковать в этот ход.

fly
NEOPLASM
НЕОПЛАЗИЯ
Каждую фазу определения ресурсов у животное, на которое сыграно, перестает действовать одно из свойств; когда все свойства перестают действовать, животное погибает.
